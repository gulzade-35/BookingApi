:hotel: Booking API – Hotel Reservation .NET CORE Projesi

Bu proje, otel arama, otel detay görüntüleme, yorum ve puanlamalar, oda bilgileri gibi özellikleri içeren bir .NET Core tabanlı otel rezervasyon sistemi örneğidir.
Proje, dış bir Booking API servisinden aldığı verileri kullanıcıya modern bir arayüzle sunar.

:rocket: Özellikler

:mag_right: Otel Arama

* Şehir, giriş–çıkış tarihleri, kişi ve oda sayısına göre arama
* API’dan gelen otellerin listelenmesi
* Her otel kartında:
   * İsim
   * Fotoğraf
   * Puan ve yorum özeti
   * "Otel Detaylarını Gör" butonu
 
:hammer: Kullanılan Teknolojiler

* ASP.NET CORE 6.0
* HttpClient ile API tüketimi
* Model sınıfları:
   * BookingApiViewModel
   * HotelDetailViewModel (Detay sayfası için)
* HTML, CSS, Bootstrap
* ThemeWagon Deluxe Teması
* Razor View + ViewComponent
