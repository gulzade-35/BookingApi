:hotel: **Booking API – Hotel Reservation .NET CORE Projesi**
_____________________________________________________________________________________________________________________________________________________________________

Bu proje, otel arama, otel detay görüntüleme, yorum ve puanlamalar, oda bilgileri gibi özellikleri içeren bir .NET Core tabanlı otel rezervasyon sistemi örneğidir.
Proje, dış bir Booking API servisinden aldığı verileri kullanıcıya modern bir arayüzle sunar.

:rocket: **Özellikler**
______________________________________________________________________________________________________________________________________________________________________

**Ana Sayfa**

 * Şehir seçimi
 * Giriş ve Çıkış Tarihi
 * Yetişkin Sayısı
 * Oda Sayısı
 * "Otel Ara" butonu

 Girilen kriterlere göre oteller listelenirç Liste Sayfasında:
 
  * Otel Adı
  * Otel Fotoğrafı
  * Otel gecelik ücreti
  * Detaylara Git butonu yer alır.
  * 
**Detay Sayfası**

Her Otelin detay sayfasında:
  * Otele ait fotoğraflar
  * Açıklama
  * Değerlendirme ve puanlar
  * Yorumlar
 
:hammer: **Kullanılan Teknolojiler**
___________________________________________________________________________________________________________________________________________________________________________________________

|Katman / Amaç    | Teknoloji                   |
|-------------    |------------------           |
| Backend         | 	ASP.NET Core 6.0          |
| API Tüketimi    | HttpClient / RapidAPI       |
| JSON İşleme     | Newtonsoft.Json             |
| Arayüz          | HTML, CSS, Bootstrap        |
| Tema            | ThemeWagon Deluxe Teması    |
| View Yapısı     | Razor View + ViewComponent  |
